# PRD — AI 研究小组展示页

## 一句话

建设「中科国光 · AI 研究小组」公开介绍页，展示团队定位、职责、核心产品、愿景与加入方式。

## 受众

- 潜在合作方与访客：快速理解团队方向与产品矩阵
- 候选人：了解培养计划与工作方式
- 内部协作：以文档与内容文件作为可维护的单一事实来源

## 信息架构（单页锚点）

1. **Hero**：品牌名（双行大字标）、口号、定位、双 CTA、公司署名与滚动提示；桌面端右侧量子核动态视觉；背景为 LLM 经典论文英文 token 生成流（等宽、渐隐、不喧宾夺主）
2. **团队介绍**：使命、高校背景、诙谐用户身份段落（荧光笔注记）
3. **职责介绍**：五个研究方向 + 核心目标
4. **核心产品**：小光、夸米、玄幂、量库、AI 课堂、引墨（图标 / 标签 / 简介）
5. **发展愿景**：量子 × AI 双向协同与未来产业交叉融合
6. **加入我们**：培养三阶段（时间线）、文化亮点、联系说明与深蓝 CTA 横幅（含团队 GitHub 入口）
7. **Footer**：公司信息、导航、GitHub

## 内容口径（事实来源）

- 公司：北京中科国光量子科技有限公司（国光量子 / GGQUANTA）
- 团队口号：让量子科研更可及，让 AI 在硬核科学中可信赖
- 产品介绍以 `Documents/product-intro/AI研究小组核心项目介绍.md` 与各产品专档为准
- 培养计划 / 管理方式以 Craft 团队文档为准
- 高校背景表述：核心成员来自北大、人大等重点高校（概括性，不罗列个人隐私）

## 非目标

- 不做完整招聘系统或在线投递表单（首版）
- 不接入后端 CMS；内容以 TypeScript 数据文件维护
- 不做深色模式优先；首版为浅色方案
